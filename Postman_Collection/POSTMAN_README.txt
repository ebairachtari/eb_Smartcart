Το αρχείο eb_SmartCart_Full_API_Flow.postman_collection.json περιλαμβάνει όλα τα API endpoints του συστήματος SmartCart, οργανωμένα ανά λειτουργική ενότητα.
Το αρχείο eb_smartcart.postman_environment.json περιέχει το απαραίτητο token environment variable για εκτέλεση προστατευμένων requests.
Για άμεση δοκιμή, να εισάγετε και τα δύο αρχεία στο Postman.
